# Myblog
#### 项目链接：www.neroarc.cn
#### 项目介绍
通过从零开始搭建整个项目，了解整个开发流程，如何设计，如何实现，一步步发现，一步步解决。系统的运用SpringBoot与它集成的各种框架，对于学习了SpringBoot却没有实际运用的人是一个很好的练手项目。

#### 主要技术与框架

  - 项目构建：Maven
  - web框架：SpringBoot
  - 数据库ORM：MyBatis
  - 数据库连接池：Druid
  - 数据库：Mysql
  - 分页插件：PageHelper
  - 数据库：MySQL
  - 模板引擎：Thymeleaf
  - markdown：Editor.md
